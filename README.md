# recruit

## Description

네카라쿠배당토 채용공고 모아보기

python selenium + bs4 패키지를 이용해 네카라쿠배당토 채용 정보를 크롤링하고, react + typescript를 이용한 웹페이지로 채용공고를 보여준다

## Dependency

- 프론트
  - node >= 20, react 18, mui 6 
  - 자세한 버전은 frontend/package.json
- 크롤러 
  - python 3.10
  - 자세한 버전은 crawl/requirements.txt
